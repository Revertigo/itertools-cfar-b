# איטרטורים - שלב ב

כתבו חמישה איטרטורים בסגנון האיטרטורים הבאים של פייתון:

* accumulate
* cycle
* filter_false
* permutation
* zip_longest

המימוש צריך לעבור את כל הבדיקות - שלכם ושלנו.

אנחנו מריצים את הפקודה הבאה:

<div dir='ltr'>

	make test && ./test
